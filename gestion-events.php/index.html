Spectacles

Interface simple pour ajouter, modifier ou supprimer les dates affichées sur la page Actus.

Mode d’emploi client

1. DéverrouillerEntrer le mot de passe de publication. Pas besoin de token GitHub au quotidien.
2. ModifierAjouter, supprimer ou corriger les dates.
3. EnregistrerCliquer sur “Enregistrer les dates sur le site”. GitHub Pages se met à jour automatiquement.

Enregistrement déverrouillé

Configuration du token permanent

À faire seulement pour installer ou remplacer le token GitHub. Le token sera stocké chiffré dans data/admin-config.json. Il ne sera pas visible en clair dans le site.

Le token déjà envoyé dans le chat doit être révoqué et remplacé par un nouveau.

Dates à afficher

Chaque ligne correspond à une date. Deux lignes avec le même titre seront regroupées sous le même spectacle.
