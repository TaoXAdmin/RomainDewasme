Spectacles

Interface simple pour ajouter, modifier ou supprimer les dates affichées sur la page Actus.

Mode d’emploi

1. TokenÀ faire une seule fois sur cet ordinateur. Le token reste dans ce navigateur.
2. DatesModifie les champs titre, date, lieu et billetterie.
3. EnregistrerClique sur “Enregistrer les dates sur le site”. GitHub Pages se mettra à jour automatiquement.

Dates à afficher

Chaque ligne correspond à une date. Deux lignes avec le même titre seront regroupées sous le même spectacle.
